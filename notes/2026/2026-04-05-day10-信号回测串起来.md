## 今天学什么

今天开始真正碰到一个最经典的量化指标：均线。

你今天的目标是：
1. 算出短均线，比如 `ma5`
2. 算出长均线，比如 `ma20`
3. 知道为什么短均线更灵敏、长均线更平滑

## 20-40 分钟任务

1. 准备一列收盘价
2. 用 `rolling().mean()` 算两条均线
3. 打印结果看看新列有没有成功出现
4. 观察前几行为什么会有空值

## 最小代码示例

```python
import pandas as pd

df = pd.DataFrame({'close': [10, 10.1, 10.3, 10.2, 10.5, 10.8, 10.7]})
df['ma3'] = df['close'].rolling(3).mean()
df['ma5'] = df['close'].rolling(5).mean()
print(df)
```
