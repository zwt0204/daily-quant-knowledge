## 今天学什么

前面几天先把脑子里的概念搭起来，今天开始把真正要用的学习环境装好。
你这一步的目标很简单：

1. 装好 Python
2. 能打开 Jupyter
3. 成功导入 `pandas`、`numpy`、`matplotlib`

今天不追求写复杂代码，只追求一件事：
**环境能正常跑起来，不报错。**

## 20-40 分钟任务

1. 安装 Python
2. 安装 Jupyter Notebook 或 JupyterLab
3. 安装下面几个库：
   - `pandas`
   - `numpy`
   - `matplotlib`
4. 打开一个 notebook 或 `.py` 文件，运行导入代码
5. 确认没有报错
