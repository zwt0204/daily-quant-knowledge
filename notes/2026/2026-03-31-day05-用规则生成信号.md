## 今天学什么

策略的关键是把观察变成规则。

1. 可以先用简单布尔条件生成信号
2. 例如短均线高于长均线，就记为持有
3. 信号是策略动作的中间结果，不是收益本身

## 20-40 分钟任务

1. 先算出短均线和长均线
2. 用条件表达式生成 `signal`
3. 看看哪些天是 1，哪些天是 0

## 最小代码示例

```python
import pandas as pd

df = pd.DataFrame({
    'close': [10.0, 10.3, 10.1, 10.5, 10.8]
})

df['ma2'] = df['close'].rolling(2).mean()
df['ma3'] = df['close'].rolling(3).mean()
df['signal'] = (df['ma2'] > df['ma3']).astype(int)
print(df)
```
