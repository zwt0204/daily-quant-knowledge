## 今天学什么

今天开始学后面写策略最常用的几种 `pandas` 操作：

1. 取列
2. 条件筛选
3. `shift`
4. `rolling`
5. `mean`

这几样东西几乎会贯穿你后面所有的量化练习。

## 20-40 分钟任务

1. 准备一列收盘价
2. 取出 `close` 这一列
3. 筛出大于某个值的行
4. 试一下 `shift(1)`
5. 试一下 `rolling(3).mean()`

## 最小代码示例

```python
import pandas as pd

df = pd.DataFrame({'close': [10, 10.2, 10.5, 10.1, 10.8]})

print(df['close'])
print(df[df['close'] > 10.2])
df['prev_close'] = df['close'].shift(1)
df['ma3'] = df['close'].rolling(3).mean()
print(df)
```
