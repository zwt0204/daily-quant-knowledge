## 今天学什么

前面你已经把价格、均线、信号、持仓、收益、净值这些零散片段都见过了。
今天的目标不是再学一个新概念，而是把这些碎片真正串成一个**完整可运行的双均线策略脚本**。

你今天要建立的是这条最小链路：

1. 先准备一列收盘价
2. 用收盘价算短均线和长均线
3. 用均线关系生成买卖信号
4. 把信号错位成真实持仓
5. 用持仓乘每日收益，算出策略收益
6. 再把策略收益累计成净值曲线

如果这 6 步你能一口气写进一个脚本里，并且运行后直接打出结果，那前面那一串内容才算真正连起来了。

## 20-40 分钟学习任务

你现在直接做这件事：

1. 新建一个 Python 脚本
2. 手动写入一小段 `close` 价格数据
3. 依次补上：`ma2`、`ma3`、`signal`、`position`、`ret`、`strategy_ret`、`equity`
4. 一次运行脚本，确认它能直接打印出结果表格
5. 用你自己的话解释下面四列分别在干什么：
   - `signal`
   - `position`
   - `strategy_ret`
   - `equity`

如果你还有余力，再多做一步：
- 把 `ma2` / `ma3` 改成别的窗口，比如 `ma3` / `ma5`，看看结果会不会变。

## 最小代码示例

```python
import pandas as pd

df = pd.DataFrame({'close': [10, 10.2, 10.5, 10.1, 10.8, 11.0, 10.7]})
df['ma2'] = df['close'].rolling(2).mean()
df['ma3'] = df['close'].rolling(3).mean()
df['signal'] = (df['ma2'] > df['ma3']).astype(int)
df['position'] = df['signal'].shift(1).fillna(0)
df['ret'] = df['close'].pct_change().fillna(0)
df['strategy_ret'] = df['position'] * df['ret']
df['equity'] = (1 + df['strategy_ret']).cumprod()

print(df[['close', 'signal', 'position', 'equity']])
```
