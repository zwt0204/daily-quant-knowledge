## 今天学什么

价格本身会涨跌，但策略分析更常用“收益率”。

1. 收益率描述的是相对变化
2. 每日收益率是最基础的量化输入之一
3. 后面的累计净值、策略收益都建立在它上面

## 20-40 分钟任务

1. 准备一列收盘价
2. 计算每日收益率
3. 看懂第一行为什么通常是空值或 0

## 最小代码示例

```python
import pandas as pd

df = pd.DataFrame({
    'close': [10.0, 10.3, 10.1, 10.5]
})

df['ret'] = df['close'].pct_change()
print(df)
```
