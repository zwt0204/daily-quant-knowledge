- 日期：2026-04-06
- 主题：定义买卖信号

## 今天学什么

有了均线之后，今天开始把“看图说话”变成明确规则。
最经典的例子就是：

- 短均线上穿长均线 → 买入
- 短均线下穿长均线 → 卖出

今天的目标不是做复杂判断，而是先把信号列写出来。

## 20-40 分钟任务

1. 准备一列收盘价
2. 算出短均线和长均线
3. 写出信号规则
4. 在表里看到 `signal` 这一列
5. 理解信号只是“想做什么”，还不是最终持仓

## 最小代码示例

```python
import pandas as pd

df = pd.DataFrame({'close': [10, 10.1, 10.3, 10.2, 10.5, 10.8, 10.7]})
df['ma3'] = df['close'].rolling(3).mean()
df['ma5'] = df['close'].rolling(5).mean()
df['signal'] = (df['ma3'] > df['ma5']).astype(int)
print(df[['close', 'ma3', 'ma5', 'signal']])
```
