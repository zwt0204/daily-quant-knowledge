## 今天学什么

围绕三件事复盘前两周的量化学习：

1. **数据从哪来**
   - 价格数据通常来自 CSV、行情接口、数据库或公开数据源。
   - 最低要求是有一列日期和一列收盘价。

2. **信号怎么生成**
   - 常见做法是把价格变成规则，例如：短期均线上穿长期均线就买入，下穿就卖出。
   - 信号本质上是把“观察”变成“条件判断”。

3. **收益怎么算**
   - 最基础先看每日收益率和累计净值。
   - 后面再扩展到手续费、滑点、仓位、最大回撤等。

## 最小代码示例

```python
import pandas as pd

# 最小样例数据
df = pd.DataFrame({
    'close': [10, 10.2, 10.5, 10.1, 10.8, 11.0, 10.7]
})

# 生成均线
df['ma2'] = df['close'].rolling(2).mean()
df['ma3'] = df['close'].rolling(3).mean()

# 生成信号：ma2 > ma3 记为持有(1)，否则空仓(0)
df['signal'] = (df['ma2'] > df['ma3']).astype(int)

# 计算日收益
df['ret'] = df['close'].pct_change().fillna(0)

# 策略收益：用昨天的仓位乘今天收益，避免未来函数
df['strategy_ret'] = df['signal'].shift(1).fillna(0) * df['ret']

df['equity'] = (1 + df['strategy_ret']).cumprod()

print(df)
```
