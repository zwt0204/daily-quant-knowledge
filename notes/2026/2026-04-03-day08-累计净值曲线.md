## 今天学什么

单日收益看不出全貌，累计净值更直观。

1. 把每日策略收益连乘起来
2. 就能得到策略资金曲线的最小版本
3. 后面看回撤、波动，都要先有净值曲线

## 20-40 分钟任务

1. 准备一列策略日收益
2. 用累计乘积算净值
3. 观察净值是如何随收益变化的

## 最小代码示例

```python
import pandas as pd

df = pd.DataFrame({
    'strategy_ret': [0.00, 0.02, -0.01, 0.03, -0.02]
})

df['equity'] = (1 + df['strategy_ret']).cumprod()
print(df)
```
