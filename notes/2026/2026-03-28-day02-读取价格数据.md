## 今天学什么

量化分析的第一步不是“建模”，而是把价格数据读进来。

1. 数据最常见格式是 CSV
2. 至少要有日期列和收盘价列
3. 先能稳定读取，再谈后面的指标和策略

## 20-40 分钟任务

1. 准备一个最小 CSV 或直接用样例数据模拟
2. 用 `pandas` 读入数据
3. 看前几行，确认读入结果符合预期

## 最小代码示例

```python
import pandas as pd
from io import StringIO

csv_text = """date,close
2026-03-24,10.0
2026-03-25,10.3
2026-03-26,10.1
"""

df = pd.read_csv(StringIO(csv_text))
print(df.head())
```
