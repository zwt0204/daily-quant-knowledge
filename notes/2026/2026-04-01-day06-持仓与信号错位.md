- 日期：2026-04-01
- 主题：持仓与信号错位

## 今天学什么

信号当天产生，但收益通常要从下一天开始吃到。

1. 这是避免未来函数的最小常识
2. 常见做法是把信号向后平移一天作为持仓
3. 不处理这个问题，结果会虚高

## 20-40 分钟任务

1. 先生成一列 `signal`
2. 再生成一列 `position = signal.shift(1)`
3. 看看两列到底差在哪

## 最小代码示例

```python
import pandas as pd

df = pd.DataFrame({
    'signal': [0, 1, 1, 0, 1]
})

df['position'] = df['signal'].shift(1).fillna(0)
print(df)
```
