- 主题：计算策略收益

## 今天学什么

有了持仓和每日收益率，就能算策略收益。

1. 先算市场每日收益率
2. 再用“昨天持仓 × 今天收益”得到策略收益
3. 这一步是回测最基础的骨架

## 20-40 分钟任务

1. 准备收盘价
2. 算出 `ret`
3. 用 `position * ret` 算出 `strategy_ret`

## 最小代码示例

```python
import pandas as pd

df = pd.DataFrame({
    'close': [10.0, 10.3, 10.1, 10.5],
    'position': [0, 0, 1, 1]
})

df['ret'] = df['close'].pct_change().fillna(0)
df['strategy_ret'] = df['position'].shift(1).fillna(0) * df['ret']
print(df)
```
