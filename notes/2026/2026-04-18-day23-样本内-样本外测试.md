# Day 23：样本内 / 样本外测试

今天这节很关键，它是昨天“过拟合意识”的下一步。

你昨天已经知道：**不能只看历史上哪组参数最赚钱。**
今天我们往前走一步，学一个真正实用的办法：

- **样本内（In-sample）**：拿来调参数的数据
- **样本外（Out-of-sample）**：调完参数后，用来检验的方法

你可以把它想成：

- 样本内 = 练习题
- 样本外 = 模拟考试

如果你把“练习题”和“模拟考试”混在一起挑最优答案，最后得出来的成绩就不可信。

## 为什么不能拿全部历史一起调参数？

因为一旦你拿全部历史一起试参数，你其实已经“偷看答案”了。

比如你试双均线参数：
- (3, 8)
- (5, 10)
- (10, 20)

如果你直接在整段历史上测试，然后选出收益最高的一组，
那这组参数其实已经利用了**未来那部分数据的信息**。

这样做的问题是：
**你并不是在检验策略能不能预测未来，而是在用未来帮你挑参数。**

这会让回测结果显得比真实情况更好。

## 一个最小流程

今天先记住最小流程：

### 第一步：切数据
把历史数据切成两段：
- 前 70%：样本内
- 后 30%：样本外

### 第二步：只在样本内调参数
比如试三组参数，看看哪组在样本内最好。

### 第三步：把“样本内最佳参数”拿到样本外验证
如果这组参数到了样本外也还行，才说明它可能不是纯粹贴历史噪声。

## 今天带你走一个直觉

假设你有一段价格序列。

你在样本内发现：
- 参数 A 收益最好
- 参数 B 一般
- 参数 C 最差

这时**不能立刻宣布 A 最强**。
你还要做一步：

把 A 拿去样本外试一下。

如果到了样本外：
- A 还是不错 → 说明它有点稳定性
- A 一下子变差 → 说明它可能只是“记住了样本内的噪声”

这一步，才是从“调参”走向“验证”的开始。

## 今天的 20-40 分钟练习

你今天不用做复杂框架，就做一个最小实验：

1. 准备一段价格数据
2. 切成前 70%、后 30%
3. 选 3 组双均线参数，比如：
   - (3, 8)
   - (5, 10)
   - (8, 15)
4. 在前 70% 上分别跑，找出表现最好的那组
5. 把这组参数拿到后 30% 上再跑一次
6. 回答两个问题：
   - 样本内最优参数，到样本外还好吗？
   - 如果我一开始就拿整段历史挑参数，为什么会高估策略效果？

## 今天最该记住的一句话

**样本内是用来选参数的，样本外是用来防止自己骗自己的。**

只会调参的人，很容易做出“历史神策略”；
会做样本外验证的人，才开始接近真正靠谱的研究流程。
